Typography it's visual representation

Typography in web design refers to the technique of organizing content that clearly communicates the intent and purpose of our products. It helps to communicate the information through visual representation. Typographic hierarchy, readability and accessibility are all vital elements of typography. Other rules to consider include using standards fonts/typefaces/colors; proper use of white space; using suitable text size, text alignment, and letter spacing. How you use typography could elevate a good user experience to a great one

Type Specifications

A typographic (or type) scale is progression of a set of fixed font sizes connected by a common ratio. It represents font sizes that work harmoniously with one another, like how a musical scale works.
